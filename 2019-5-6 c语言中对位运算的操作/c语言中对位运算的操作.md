# c语言中对位运算的操作 #

1、C语言中的 >>= 意思为：右移后赋值

代码示例为：x = 8; 
x >>= 3; 

右移后结果为：00000000 00000000 00000000 00000001



2、C语言中的 <<= 意思为：左移后赋值

代码示例为：

x = 8;

x <<= 3; 

左移后赋值结果为：00000000 00000000 00000000 01000000



3、C语言中的 &= 意思为：按位与后赋值

代码示例为：

x = 0x02; 

x &= 0x01; 

按位与后的结果为：0x00



4、C语言中的 ^= 意思为：按位异或后赋值

代码示例为：

x = 0x02; 

x ^= 0x01; 

按位异或的结果为：0x03 等同于0011



5、C语言中的 |= 意思为：按位或后赋值

代码示例为：

x = 0x02; 

x  |= 0x01;

按位或的结果为：0x03 等同于0011
